# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Priya-02/AI-Data-Annotation-Platform.git
```

Go to the project folder

```bash
cd AI-Data-Annotation-Platform
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
